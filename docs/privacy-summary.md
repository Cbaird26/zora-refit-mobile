# Zora Refit Privacy Summary

## Current data model

- Primary state is stored on device
- No account system
- No remote database
- No analytics SDK
- No advertising SDK
- No cloud sync

## Data the app can create locally

- Interface state
- Scenario inputs
- Logged run archives exported by the user

## Native capabilities currently used

- Haptics
- Preferences storage
- Local file export
- Native share sheet
- Splash screen
- Status bar styling

## Store-facing position

For listing and privacy-policy text, the current honest framing is:

`Zora Refit stores app state locally on device and only shares exported files when the user explicitly chooses to export/share them.`
